The Typeahead is basically a component which:

  • takes a user input as a string
  • pipes this input to an Observable chain to get a list of options
  • handles the display of the popup containing the options
  • handles the navigation through the options
  • handles the selection of one option

Graphically it's a combination of:

  • an input used to:
    • edit the search term
    • display the selected option
  • a popup displaying the list of options

The Typeahead is basically a component which:

  • takes a user input
  • pipes this input to an Observable chain to get a list of options
  • handles the popup of options and option selection

Graphically it's a combination of an input and a popup containing a list.

Let's introduce some concepts:

  • the search term: is used as input to produce a result list
  • the result list, and the result item: the list given to the Tyeahead for display
  • the options list, and the option item: the list displayed by the Typeahead in its attached popup, coresponding 1 by 1 to the result list
  • the input field: holds a representation of the currently selected result item if any, and allows editing the search term
  • the popup: the popup attached to the input field and displaying the options list

The Typeahead is basically a component which:

  • takes a user input
  • pipes this input to an Observable chain to get a list of options
  • handles the popup of options and option selection

Graphically it's a combination of an input and a popup containing a list.

Let's introduce some concepts:

  • the search term: a string used as input to produce a results list
  • the results list and the result item: the list given to the Typeahead for display; goes through formatting before to be converted to an options list for display
  • the options list and the option item: the list displayed by the Typeahead in its attached popup, corresponding 1 by 1 to the results list
  • the input field: holds a formatted representation of the currently selected result item if any, and allows editing the search term
  • the popup: the popup attached to the input field and displaying the options list

For the sake of the demos, the Typeahead instances will filter out a basic list of colors, keeping only the ones starting with the search term.

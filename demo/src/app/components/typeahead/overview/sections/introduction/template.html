Typeahead example:

The Typeahead is basically a component which:

  • takes a user input as a string: try typing bl
  • pipes this input to an Observable chain to get a list of options
  • handles the display of the popup containing the options: you should see the colors black and blue
  • handles the navigation through the options: try using the keyboard arrows up and down
  • handles the selection of one option: press enter to select the highlighted option, or click directly on it, and see it appear in the input field
